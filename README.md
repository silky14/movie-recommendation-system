# Movie-Recommendation-
An end-to-end Machine Learning based Movie Recommendation System that suggests similar movies using content-based filtering. The project leverages TF-IDF Vectorization and Cosine Similarity on movie metadata to generate accurate recommendations through an interactive Streamlit web application.



* Built a content-based Movie Recommendation System using Machine Learning techniques.
* Applied TF-IDF Vectorization to transform movie metadata into numerical feature representations.
* Used Cosine Similarity to identify movies with similar content and generate personalized recommendations.
* Developed a complete ML pipeline including data preprocessing, feature engineering, model generation, and recommendation serving.
* Created an interactive Streamlit web application for real-time movie recommendations.
* Optimized recommendation retrieval using precomputed similarity matrices for fast inference.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-learn
* TF-IDF Vectorizer
* Cosine Similarity
* Joblib
* Streamlit

## Machine Learning Workflow

1. Data Collection (TMDB 5000 Movie Dataset)
2. Data Cleaning and Preprocessing
3. Feature Engineering using genres, cast, crew, keywords, and overview
4. TF-IDF Vectorization
5. Cosine Similarity Matrix Generation
6. Recommendation Engine Development
7. Interactive Streamlit Deployment

## Key Learning Outcomes
